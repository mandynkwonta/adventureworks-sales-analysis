# adventureworks-sales-analysis
Sales Analysis dashboard using power BI
## AdventureWorks Sales Analysis

**Tools:** Power BI, DAX, Data Modeling

### 📌 Project Overview

This project analyzes sales data from AdventureWorks to uncover insights into revenue, customer behavior, product performance, and regional trends.

### 🔧 What I Did

* Cleaned and transformed raw data
* Built relationships across multiple tables
* Created KPIs like Total Revenue, Total Orders, and Average Order Value
* Designed a 5-page interactive dashboard

### 📊 Key Insights

* Revenue is concentrated in top-performing products
* Certain regions outperform others significantly
* Customer behavior shows repeat purchase patterns
